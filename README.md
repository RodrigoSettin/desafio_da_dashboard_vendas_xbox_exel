# Dashboard de Vendas Xbox em Excel

Projeto desenvolvido como parte de um desafio da Digital Innovation One (DIO), com foco na criação de um dashboard interativo em Excel para análise de vendas e assinaturas do Xbox.

## Objetivo

Analisar dados de assinaturas Xbox, permitindo a visualização de:
- Receita total
- Distribuição de planos
- Tipos de assinatura
- Serviços adicionais
- Uso de cupons

O dashboard transforma dados brutos em informações visuais para apoio à tomada de decisão. 

## Arquivo do Projeto

O repositório contém o arquivo:

- Dados_para_dastbord_vendas_Xbox_04-01-26 REV3.xlsx

O arquivo possui base de dados, cálculos auxiliares e o dashboard final com gráficos e segmentações interativas.

## Dados Utilizados

Os dados utilizados foram fornecidos pela Digital Innovation One (DIO) como parte do desafio, sendo utilizados exclusivamente para fins educacionais, simulando vendas e assinaturas de planos Xbox.

## Como Utilizar

1. Faça o download do arquivo Excel
2. Abra no Microsoft Excel (versão 2016 ou superior)
3. Habilite o conteúdo, se solicitado
4. Acesse a aba do dashboard e utilize os filtros disponíveis
